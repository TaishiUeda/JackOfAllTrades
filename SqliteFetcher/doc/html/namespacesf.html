Constructor

Setting data types

Column_t col = {
{"ID", Data(INT64, PRIMARY_KEY)},
{"First name", Data(TEXT)},
{"Second name", Data(TEXT)},
{"Age", Data(INT32)},
{"Height_cm", Data(REAL)},

Setting data values and each type is detemined automatically.

{"ID", Data(0, PRIMARY_KEY)},

Column_t col;
